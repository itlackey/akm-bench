# Skill Library Evolution

## From Prompts to Assets

The evolution of agentic skill libraries follows a predictable pattern:

1. **Ad-hoc prompts** — Copy-pasted text snippets
2. **Prompt templates** — Parameterized templates with placeholders
3. **Structured skills** — Typed assets with metadata and discoverability
4. **Composable agents** — Skills, commands, and agents that reference each other

## Design Principles

- **Discoverability** — Assets should be findable via natural language search
- **Composability** — Small, focused assets that combine well
- **Portability** — Assets work across different AI runtimes

## Metadata Matters

Rich metadata (tags, descriptions, search hints) dramatically improves retrieval quality.
